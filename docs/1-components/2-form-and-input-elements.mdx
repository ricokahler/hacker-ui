# Form and Input Elements

## Form Control Basics

Form Controls give form elements context.

You can place elements like `<Label />`s, `<TextInput />`s, and `<HelperText />`s inside `<FormControl />`s to synchronize their focus, error, and disabled states for one field.

## Radio Buttons

You radio buttons with `<RadioGroup />`s. A `<RadioGroup />` automatically sets the `name` for each form element.

You may also pass in a `value` and `onChange` handler to the `RadioGroup` instead of to each individual `Radio` button.
